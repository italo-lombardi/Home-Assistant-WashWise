# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅        |
| < 0.1   | ❌        |

## Reporting a Vulnerability

Please report security vulnerabilities via [GitHub Private Security Advisory](https://github.com/italo-lombardi/Home-Assistant-WashWise/security/advisories/new).

Do not open a public issue for security vulnerabilities. You can expect an initial response within a few days. If the vulnerability is accepted, a fix will be released and credit given upon disclosure.
